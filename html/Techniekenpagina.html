Technieken

HTML

Ik heb gebruik gemaakt van verschillende HTML-elementen zoals <div> voor structuur, <header> voor de kop van de pagina en <ul> voor lijsten.

CSS

Voor de opmaak heb ik CSS gebruikt om de kleuren, lettertypes en de layout aan te passen. Ik heb Grid toegepast voor een responsieve indeling en CSS animations gebruikt zodat de title een mooie kleur krijgt.

JavaScript

Met JavaScript heb ik interactieve elementen toegevoegd. Zo verandert de titel wanneer erop wordt geklikt met de functie veranderTitel(), en verschijnt er een bericht wanneer je op een knop klikt met toonBericht().
